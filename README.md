# 📊 Amazon Sales Performance Dashboard

## 📌 Project Overview

This project is an interactive Power BI dashboard developed to analyze Amazon sales performance. The dashboard helps business users monitor sales, profit, orders, product performance, and monthly sales trends through interactive visualizations and filters.

---

## 🎯 Business Objective

The objective of this dashboard is to help decision-makers:

- Analyze sales performance
- Monitor profit and profit margin
- Identify top-selling products
- Track monthly sales trends
- Compare sales across categories
- Analyze sales using interactive filters

---

## 📊 Business Insights

- Storage category generated the highest sales.
- Sales reached the highest value in December.
- Top 10 products contributed significantly to total revenue.
- Profit margin remained around 11%.
- Interactive filters allow users to analyze data by date, category, and city.

## 📌 Data Modeling

A Star Schema was used in this project.

Fact Table:
- Sales

Dimension Tables:
- Product
- Category
- Customer
- Date
- City

Relationships:
- One-to-Many relationships between dimension tables and the Sales fact table.
