# 📊 Amazon Sales Performance Dashboard

## 📌 Project Overview

This project analyzes Amazon sales data using Microsoft Excel and Power BI. The data was first prepared and cleaned in Excel, then imported into Power BI for data modeling, DAX calculations, and interactive dashboard development.

---

## 🛠 Tools & Technologies

- Microsoft Excel
- Power BI Desktop
- Power Query
- DAX
- Data Modeling
- Data Visualization

---

## 🔄 Project Workflow

1. Collected the Amazon sales dataset in Microsoft Excel.
2. Cleaned and validated the data in Excel.
3. Imported the dataset into Power BI.
4. Performed additional data transformation using Power Query.
5. Created a Star Schema data model with fact and dimension tables.
6. Developed DAX measures for KPIs.
7. Built an interactive dashboard with charts, slicers, and KPI cards.
8. Generated business insights to support decision-making.

---

## 📊 Data Preparation

### Microsoft Excel
- Checked for missing values
- Removed duplicate records
- Verified data consistency
- Organized the dataset into a structured format

### Power Query
- Changed data types
- Renamed columns
- Removed unnecessary columns
- Prepared data for analysis

---

## 📌 Data Modeling

A Star Schema was implemented.

### Fact Table
- Sales

### Dimension Tables
- Customer
- Product
- Category
- Date
- City

### Relationships
- One-to-Many relationships between dimension tables and the Sales fact table.

## 📊 Dashboard Features

## 💡 Business Insights

- Storage category generated the highest sales.
- Monthly sales peaked in December.
- Top-selling products contributed significantly to overall revenue.
- Profit margin remained above 11%.
- Interactive filters allow users to analyze sales by date, category, and city.
